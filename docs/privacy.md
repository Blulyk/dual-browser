# Privacidad de Dual Browser

Dual Browser 0.1 no incluye cuentas, telemetría, publicidad ni un backend propio.

## Datos locales

La aplicación guarda en el dispositivo pestañas normales, historial, favoritos y preferencias. Las pestañas privadas no se añaden al historial ni a la sesión restaurable. Android System WebView y el proveedor de búsqueda o los sitios visitados mantienen sus propias políticas de privacidad.

## Permisos

Internet es necesario para navegar. Cámara, micrófono y ubicación se solicitan mediante Android solo cuando una página los pide. El selector de archivos permite al usuario elegir explícitamente qué archivo entregar. Las descargas se envían al gestor de descargas de Android.

## Diagnóstico

Diagnostics muestra modelo, versión Android/WebView y características de las pantallas. La aplicación no lo transmite automáticamente; solo se comparte al pulsar **Export** y elegir un destino en Android.

## Borrado

Los datos pueden eliminarse borrando el almacenamiento de Dual Browser desde Ajustes de Android o desinstalando la aplicación.
